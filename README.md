# Emotion Predictor using NLP

This project is a Natural Language Processing (NLP) based Emotion Predictor that analyzes user input text and predicts the emotion expressed in the sentence.

The application uses Machine Learning with Scikit-learn, TF-IDF Vectorization, and a Flask web interface to provide real-time emotion predictions.

Users can enter any sentence in the web interface and the model predicts the emotion behind the text.

---

## Features

- Predicts emotion from user input text
- Uses NLP techniques for text processing
- Machine learning model trained using Scikit-learn
- Flask backend for handling predictions
- Simple web interface using HTML, CSS and JavaScript
